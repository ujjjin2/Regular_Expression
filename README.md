# 정규표현식 
#### ■ 정규표현식이 필요한 이유
정규 표현식이란?
- 복잡하고 특정한 규칙을 가진 문자열을 처리할 때 사용하는 기법
- 복잡한 문자열의 검색과 치환을 위해 사용됨
- 문자열을 처리하는 모든 곳에서 사용
- 정규표현식을 정규식이라고도 한다. 

#### ■ 정규표현식이 필요한 이유
- 정규식을 사용하면 직관적인 코드를 작성할 수 있다.
- 간결한 코드 작성이 가능하다.
